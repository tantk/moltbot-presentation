üéØ Philosophy 07 / 24

Core Design Principles

1. Local-First

The Gateway is a single control plane running on your device. Sessions, channels, tools, and events all managed locally.

2. Multi-Agent Routing

Route inbound channels to isolated agents with per-agent sessions and workspaces for different contexts.

3. Always-On Voice

Voice Wake + Talk Mode for macOS/iOS/Android with ElevenLabs integration. Speak naturally to your assistant.

4. Live Canvas

Agent-driven visual workspace with A2UI. See what the AI is thinking, browsing, or creating in real-time.

5. Skills Platform

Extensible through bundled, managed, and workspace skills. Install capabilities like packages.

6. Security by Default

DM pairing required for unknown senders. Explicit opt-in for public inbound messages.

üîí Security 12 / 24

Security by Default

"Moltbot connects to real messaging surfaces. Treat inbound DMs as untrusted input."

‚Äî Security Documentation

DM Pairing

Unknown senders receive a pairing code. The bot does not process their message until explicitly approved.

# Approve a pairing request
moltbot pairing approve <channel> <code>

Security Policies

  • dmPolicy="pairing" ‚Äî Require approval (default)
  • dmPolicy="open" ‚Äî Accept all (opt-in)
  • allowFrom ‚Äî Whitelist specific users
  • Block list ‚Äî Deny specific senders

ü©∫ Doctor Command

Run moltbot doctor to surface risky or misconfigured DM policies. Recommended for periodic security audits.
